# Interactive Explorer

Three views into the series:

- **Algebraic Structures**: the hierarchy from semigroup through fields, lattices, and morphisms. Click any node for its laws, C++20 concept, and code.
- **Posts**: all nineteen posts as a network, connected by shared algebraic structures.
- **Structure → Algorithm**: the hierarchy annotated with what algorithm each structure enables.

Drag nodes to rearrange. Click to see details. Use the search box to filter.
