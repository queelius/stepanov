## Building

```bash
# Configure and build
make build

# Run all tests
make test

# Build documentation
make docs

# Serve documentation locally
make docs-serve
```

## Contributing

Contributions are welcome. Each post should:

- Demonstrate one core idea
- Be self-contained (~100-400 lines)
- Include tests
- Follow the algebraic design pattern
